# Backend Service
**Планируются следующие сервисы:**

- **backend** - главный сервис, общающийся с фронтендом и другими сервисами
- **speech-to-text** - сервис, генерирующий текст из голоса
- **translation** - сервис, отвечающий за машинный перевод
- **text-to-speech** - сервис, генерирующий голос из текста
